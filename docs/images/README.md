# スクリーンショット配置場所

このディレクトリにアプリケーションのスクリーンショットを保存してください。

### 必須

1. **pharmacist-dashboard.png** - 薬剤師ダッシュボード
   - カレンダー表示
   - シフト希望登録フォーム
   - 確定シフト一覧

2. **pharmacy-dashboard.png** - 薬局ダッシュボード
   - カレンダー表示
   - シフト募集登録フォーム
   - 確定シフト一覧

3. **admin-dashboard.png** - 管理者ダッシュボード
   - 全体のカレンダービュー
   - マッチング機能
   - 統計情報

4. **login-general.png** - 一般ログイン画面
   - 薬剤師・薬局用のログイン画面

5. **login-admin.png** - 管理者ログイン画面
   - `/admin-login`の画面

### 推奨（追加すると良い）

6. **ai-matching.png** - AIマッチング実行中の画面
7. **calendar-view.png** - カレンダー詳細ビュー
8. **profile-edit.png** - プロフィール編集画面
9. **shift-confirmed.png** - 確定シフト詳細
10. **mobile-view.png** - モバイル表示

## 撮影方法

### デスクトップ

1. アプリケーションをブラウザで開く
2. デベロッパーツールで解像度を1920x1080に設定
3. スクリーンショットを撮影（Cmd+Shift+4 または 画面キャプチャ）
4. このディレクトリに保存

### モバイル

1. デベロッパーツールでモバイルビュー（390x844）に切り替え
2. スクリーンショットを撮影
3. ファイル名に `-mobile` を付けて保存（例: `pharmacist-dashboard-mobile.png`）

## ファイル命名規則

- 小文字とハイフンを使用
- 拡張子は `.png` を推奨（高品質）
- 例: `admin-dashboard.png`, `login-general.png`

## 画像サイズ

- PNG形式推奨
- 最大幅: 1920px
- 適度に圧縮（TinyPNG などを使用）
